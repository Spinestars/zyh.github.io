nginx ☞ 泛域名_变量截取
2018-09-06|web

location 优先级从上往下依次递减：
location = 仅匹配字符串自身
location ^~ 匹配某个字符串开头的uri
location ~ 正则匹配，区分大小写
location ~* 正则匹配，不区分大小写
location / 表示匹配“域名/之后的uri”，再比如localtion /images，表示匹配“域名/images之后的uri

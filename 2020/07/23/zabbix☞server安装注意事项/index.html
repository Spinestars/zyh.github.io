zabbix ☞ server安装注意事项
2020-07-23|监控

  1. 数据库编码一定要安装官方来命令来设置，例如zabbix5编码是 utf8和utf8_bin

  2. 最后web控制台安装界面，要确保zabbix-server可以访问设置的 hostname 的 10051 端口。如果hostname写的域名，那么要确保zabbix-server可以通过 hosts本地解析或者外网访问10051端口

  3. zabbix-server 的 agent 要使用拉取模式，因为官方给的模板无法直接修正为推流模式
